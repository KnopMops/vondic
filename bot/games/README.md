# Игры для ботов (HTML/CSS/JS)

Архив ZIP с `index.html` в корне загружается через API или кнопку «Загрузить игру» (проверка на платформе).

Играть пользователь должен **в чате с ботом**, к которому привязана игра: бот присылает inline-игру через Botik API (`send_game`).

## 2048 (пример)

```bash
cd bot/games/2048
zip -r ../2048.zip .
```

Загрузите `2048.zip` для вашего бота (`POST /api/v1/bots/{bot_id}/games`).

В чате с ботом: **Играть** → список → выбор игры → iframe в сообщении.
